# Kolmogorov-Arnold-Network (KAN)

### **Kolmogorov Arnold Network(KAN) + MobileNetV2**

| Model                  | Val Accuracy | Val Loss | Model Size  |
|------------------------|--------------|----------|-------------|
| MobileNetV2 + MLP FC   | 0.9744       | 0.3373   | 10.45 MB    |
| MobileNetV2 + KAN FC   | 0.9750       | 0.0631   | 22.3 MB     |
